# Modern Landing Pages Collection

A collection of high-performance, responsive, and aesthetically pleasing landing pages built with modern web technologies. This project features a "Cyber-Elegant / Ultra-Futuristic" design theme with deep obsidian backgrounds, vibrant gradients, and glassmorphism effects.

## Project Overview

This repository contains multiple landing page templates designed for various use cases, including:
- **QuantumGrid (index.html):** A next-gen cloud infrastructure platform landing page.
- **Aurora (index1.html):** AI-powered digital marketing platform.
- **NexusFlow (index2.html):** SaaS product management and workflow tool.
- **PixelForge (index3.html):** Creative design and AI art generation tool.
- **Faiz Digital (index4.html & landing.html):** Portfolio and service showcase for a digital marketing expert.
- **Aakash Cloud (index5.html):** Next-gen Indian cloud infrastructure with a **Neumorphic Light (Soft UI)** dashboard layout.
-   **SkyGlass (index6.html):** Premium Glassmorphic Cloud Landing Page with animated gradient orbs and floating UI.
-   **Garuda Gaming (index7.html):** Extra Ordinary Indian Esports Ecosystem with 3D interactions.
- **Authentication Pages:** Modern Login and Sign Up pages with validation and animations.

## Features

- **Responsive Design:** Fully responsive layouts that look great on all devices (Mobile, Tablet, Desktop).
- **Modern UI/UX:**
  - Glassmorphism effects
  - Animated backgrounds (Tech grids, glowing orbs, gradients)
  - Smooth scroll animations
  - Interactive elements
- **Iconography:** All icons have been migrated to **Lucide Icons** for a crisp, consistent, and lightweight visual language.
- **Animations:**
  - CSS keyframe animations (fade-in, slide-up, pulse, float)
  - Spin animations for loading states
  - Scroll-triggered reveals
- **Tech Stack:**
  - HTML5
  - CSS3 (Custom properties, Flexbox, Grid)
  - Vanilla JavaScript
  - [Lucide Icons](https://lucide.dev/) (v0.344.0+)

## Setup & Usage

No build process is required. These are static HTML pages.

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
2.  **Open any file:**
    Simply open the `.html` files (e.g., `pages/index.html`) in your preferred web browser.
